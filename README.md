# 🌐 Limbus Dungeon Guide (림버스 컴퍼니 거울 던전 가이드)

> **림버스 컴퍼니(Limbus Company)의 거울 던전 팩 선택지와 에고 기프트(E.G.O Gift) 정보를 한눈에 확인하고 검색할 수 있는 웹 서비스입니다.**

---

## 📌 프로젝트 소개 (Project Overview)
림버스 컴퍼니의 핵심 콘텐츠인 **'거울 던전'**을 진행할 때, 어떤 팩을 선택해야 유리할지, 어떤 에고 기프트가 내 덱에 맞을지 고민하는 수감자(유저)들을 위해 제작되었습니다. 

매번 바뀌는 던전 속에서 빠르고 정확하게 정보를 검색하여 효율적인 거울 던전 파밍을 도와줍니다.

- **배포 링크:** [👉 여기에 웹사이트 배포 URL 입력 (e.g., GitHub Pages, Vercel 등)](https://your-link.com)

---

## ✨ 주요 기능 (Key Features)

* **📦 던전 팩(Theme Pack) 정보 제공**
    * 각 거울 던전 팩별 출현하는 적, 테마, 보스 정보 요약
    * 팩 선택 시 등장하는 이벤트 및 선택지 결과 정리
* **🎁 에고 기프트(E.G.O Gift) 도감 및 검색**
    * 키워드(오만, 질투, 화상, 출혈, 침잠 등) 및 이름 기반 실시간 검색
    * 각 에고 기프트의 등급, 효과, 강화(동기화) 정보 시각화
* **⚡ 키워드 필터링**
    * 현재 자신이 조합 중인 인격/에고 덱에 맞는 기프트를 쉽게 찾을 수 있는 필터 기능
* **📱 반응형 웹 디자인**
    * 모바일 환경에서도 거울 던전을 플레이하며 한 손으로 쉽게 검색할 수 있는 모바일 최적화 레이아웃
